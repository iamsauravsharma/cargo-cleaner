cargo-trim-config 0.6.1
Query config file data

USAGE:
    cargo trim config [FLAGS]

FLAGS:
    -f, --file         Return config file location
    -d, --directory    Query about directory data
    -e, --exclude      Query about exclude data
    -h, --help         Prints help information
    -i, --include      Query about include data
    -p, --print        Print/Display config file content
    -V, --version      Prints version information
